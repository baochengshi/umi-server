aliyun fc SSR using Umi

[Demo 预览](http://umi.ssr-fc.com)

## Usage

先安装依赖，

```js
$ yarn
```

### 开发

```sh
$ yarn dev
```

### 构建

```sh
# 构建前后端
$ npm run build
# 构建后端
$ npm run build:fc
# 构建前端
$ npm run build:umi
```

### 部署

```sh
$ npm run deploy
```
